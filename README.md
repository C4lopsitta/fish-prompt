# Fish Prompt
A simple custom Fish Prompt that uses [Rose Pine](https://rosepinetheme.com/) as a Color Palette.

## Installation
1. Clone the repository
```sh
$ git clone https://github.com/C4lopsitta/fish-prompt.git
```
2. Enter the folder
```sh
$ cd fish-prompt
```
3. Run the installation script
```sh
$ ./install.sh
```
